# Reference Videos

Use this file for mechanism and packaging inspiration. Follow `docs/ARCHITECTURE.md` and `cad/DIMENSIONS.md` for the actual design.

## Actionable

### 1. Conveyor packaging

- [MINI Conveyor belt for school project](https://www.youtube.com/watch?v=PA9kk9B5rEU) by Maker Tutor
  Primitive conveyor example. Use it to study compact conveyor layout and basic timing-belt packaging.

### 2. Stepper conveyor drive

- [How to Make 2-Way Conveyor Belt System Using Stepper Motor & Arduino](https://www.youtube.com/watch?v=qoejrZhn5XI) by Easy HomeMade Projects
  Use it as a reference for `stepper motor -> timing belt -> belt roller` packaging and motor placement. See `reference/EASY_HOMEMADE_PROJECTS_CONVEYOR_MECHANICS.md` for the modified states-build mechanics notes.

### 3. Conveyor dimensions

- [How to Make a Conveyor Belt (PDF Plans)](https://www.youtube.com/watch?v=PjL9drjZAQU) by Construeye Facil
  Use it to sanity-check roller, frame, and belt proportions while building our CAD.

### 4. Selector chute design

- [3D Printed Color Sorting Machine | Arduino and 3D print Project](https://www.youtube.com/watch?v=vOLTPEVGzjM) by Carlos 3D FR
  Use it for chute-based sorting inspiration and to study how a custom mechanism gets iterated.

### 5. Basic sorter flow

- [Smart Conveyor using Arduino ESP32](https://www.youtube.com/watch?v=HdYP6mvLWew) by Hacktuber
  Use it as a simple reference for overall sorter flow, layout, and sequence.

## General ideas

### 1. Diverter chute motion

- [Marble Lift + Marble Sorter](https://www.youtube.com/watch?v=2tBa83m16nc) by Engineezy
  Diverter chute motion idea with useful mechanical transitions and routing movement.

### 2. Side-arm diverter

- [Linear Sorter - Linear Side Arm Sorter](https://www.youtube.com/watch?v=uVoZxnJS6xw) by Falcon Autotech

### 3. Rotary singulation

- [Automatic Feeder for marble sorter](https://www.youtube.com/shorts/l8QTaY-TBvg) by GStarsMedia
