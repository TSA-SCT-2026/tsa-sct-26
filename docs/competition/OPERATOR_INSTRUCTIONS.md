# Operator Instructions

> This document is the written instructions left for evaluators at conference day. Write this for someone who has never seen the system. Plain language. No jargon. Must be completable in under 2 minutes.

---

## PLACEHOLDER -- write this after the system is fully calibrated and working

The final version of this document should cover:

1. What to load and how (24 LEGO bricks, which orientation, how to stack them in the chute)
2. How to power on (which switch or button)
3. How to start a run (the single momentary button)
4. What the display shows during the run (so the evaluator knows it is working)
5. What SORT COMPLETE looks like (what the display shows at end)
6. What to do if the system halts mid-run (what the display shows, how to clear and restart)
7. How to run a second demonstration if requested

---

## Design constraints for this document

- Must fit on a single page, printed, laminated if possible
- Font large enough to read at arm's length
- No school names, no student names -- State Conference ID only
- Steps are numbered, not paragraphs
- Include a diagram or photo of the chute loading orientation if there is any ambiguity
- Tested: have someone unfamiliar with the system follow these instructions cold before conference day
