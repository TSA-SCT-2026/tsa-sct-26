# CAD

CAD files in this repository describe the finished sorter geometry.

## Files

- `conveyor_assembly.step`: conveyor assembly reference
- `assemblies/`: assembly organization notes

## Placement Rules

- Keep source CAD, exported STEP files, and printable parts in clearly named folders.
- Put active build geometry in the public tree.
- Put obsolete or experimental geometry under `_archive/`.
- Do not store slicer output, machine temp files, or OS metadata.
